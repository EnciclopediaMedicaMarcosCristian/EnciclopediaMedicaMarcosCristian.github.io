Declaración de Accesibilidad

Nos comprometemos a asegurar la accesibilidad de nuestro sitio web siguiendo las directrices WCAG 2.1.

Idioma principal

El idioma principal de todas nuestras páginas está correctamente identificado para facilitar la navegación mediante lectores de pantalla.

Título de la página (WCAG 2.4.2)

Cada página contiene un título descriptivo que refleja claramente su contenido.

Texto alternativo en imágenes (WCAG 1.1.1)

Las imágenes relevantes incluyen texto alternativo que describe su contenido o función, facilitando así la función de los lectores de pantalla.

Navegación y enlaces (WCAG 2.4.4 y 3.2.4)

Se han añadido enlaces accesibles para facilitar la navegación entre secciones y páginas. La imagen de la cabecera enlaza a la página principal.

Estructura y etiquetas semánticas (WCAG 1.3.1)

La estructura de las páginas sigue una jerarquía clara de encabezados y el etiquetado semántico correcto, facilitando el acceso a la información.

Formularios (WCAG 3.3.1)

Los formularios incluyen identificación de errores e instrucciones claras para el usuario.

Contraste (WCAG 1.4.3)

Se mejoró el contraste entre el texto (#333) y el fondo (#f4f4f4) para cumplir con un mínimo de 4.5:1, siguiendo las pautas WCAG.

Enfoque visible (WCAG 2.4.7)

Se añadió un estilo de foco visible con borde negro y cambio de color de fondo en elementos interactivos, asegurando que el enfoque sea claramente visible al navegar con teclado.

Orientación (WCAG 1.3.4)

El diseño es flexible y adaptable, con anchos relativos para asegurar una visualización adecuada tanto en dispositivos móviles como en pantallas de escritorio.

Uso de color (WCAG 1.4.1)

Se usaron múltiples indicadores, como el color y los íconos, para no depender exclusivamente del color como información.

Tamaño del texto (WCAG 1.4.4)

El uso de unidades relativas para los elementos de texto permite que el texto sea redimensionable sin pérdida de contenido o funcionalidad.

Control del espaciado (WCAG 1.4.12)

Se proporcionó suficiente espacio entre los campos y los elementos de texto para evitar que se superpongan o dificulten la lectura cuando se aumente el tamaño del texto o espaciado.

Navegación mediante teclado (WCAG 2.1.1)

Todos los elementos interactivos pueden ser accedidos y utilizados mediante el teclado, asegurando accesibilidad para usuarios con limitaciones motrices.

Formularios accesibles (WCAG 3.3.2)

Se usaron etiquetas claras y visibles para todos los campos de formulario, lo que mejora la accesibilidad para lectores de pantalla.

Enlaces "saltar a" (WCAG 2.4.1)

Los enlaces “saltar a” ayudan a mejorar la accesibilidad para personas que usan el teclado o tienen movilidad reducida, permitiendo un acceso directo al contenido principal.

Media Queries (WCAG 1.4.10)

Se implementaron media queries para asegurar una visualización adecuada en pantallas de baja resolución (480px), reorganizando el contenido sin pérdida de funcionalidad o legibilidad.

Fuente (WCAG 1.4.4)

Se utilizó una fuente legible y escalable, permitiendo que el texto se pueda aumentar hasta un 200% sin pérdida de funcionalidad o contenido.
